# Требования к кваду

* возможность дистанционного управления
 * с пульта
 * с телефона (??)
* автоматическое отслеживание заданного объекта по данным от видеокамеры (Tracking-learning-Detection algorithm)
* автоматическое следование за заданным объектом по данным трекера
* камера высокого  разрешения для съёмки видео (HDcam)
* поворотный подвес для HDcam
* камера низкого разрешения для получения потока в TLD алгоритм (TLDcam)
* сенсор (камера?) для избежания столкновений во время движения (Spidycam)
* алгоритм умной камеры для автоматического определения нужного плана, угла съёмки, дистанции (чтобы круто смотрелось снимаемое)
* возможность аварийной посадки
* скорость горизонтального движения до 60 км/ч (???? какие ограничения вообще есть?)
